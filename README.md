# Password Generator
[![License: Unlicense](https://img.shields.io/badge/license-Unlicense-blue.svg)](http://unlicense.org/)

## Description

The Password Generator prompts the user to provide criteria required for the generated password, then displays a randomly generated password that meets those criteria.
